# TASK-T01 小白教學：任務資料

## 這個功能做什麼？

`tasks.json` 像一張全組共用的工作清單。每筆任務會寫明負責人、狀態、會修改哪些檔案，以及做到什麼程度才算完成。

```text
tasks.json
  ↓ 讀取與檢查
task_loader.py
  ↓
任務中心畫面
```

若新增任務，請複製現有物件並更換 `task_id`，不要刪除既有欄位。狀態只能使用：待認領、進行中、等待測試、需要修改、已完成、已封存。

## 怎麼測試？

執行 `python -m pytest tests/test_task_loader.py -q`。若 JSON 少欄位，測試或頁面會顯示缺少的欄位名稱。
